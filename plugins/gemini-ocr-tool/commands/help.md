Display help information for gemini-ocr-tool CLI commands.

## Usage

```bash
# Show general help
gemini-ocr-tool --help

# Show command-specific help
gemini-ocr-tool COMMAND --help

# Show version
gemini-ocr-tool --version
```

## Arguments

- `COMMAND` (optional): Specific command to get help for
- `--help` / `-h`: Show help information
- `--version` / `-v`: Show version information

## Examples

```bash
# General help
gemini-ocr-tool --help

# Command help
gemini-ocr-tool search --help

# Version information
gemini-ocr-tool --version
```

## Output

Displays usage information, available commands, and options.
